httpd配置
./configure --prefix=/usr/local/http2 \
--enable-modules=all \
--enable-mods-shared=all \
--enable-so
	可以识别so后缀的模块文件

--enable-mods-shared=all \
	模块共享型：shared  共享类型
	要把全部的“功能模块”代码编译到“Apache本身软件”内部
特点：
Apache本身软件会稍显臃肿，但是在调用相关模块的时候速度很快

相对应类型：static 静态类型
Apache内部要被编译进许多模块代码，其他模块代码都独立存在，需要什么模块，就立即include引入之
特点：
Apache软件本身很小巧，其运行速度很快


php配置：
./configure --prefix=/usr/local/php \
--with-apxs2=/usr/local/http2/bin/apxs \
--with-mysql=mysqlnd \
--with-pdo-mysql=mysqlnd \
--with-mysqli=mysqlnd \
--with-freetype-dir=/usr/local/freetype \
--with-gd=/usr/local/gd \
--with-zlib --with-libxml-dir=/usr/local/libxml2 \
--with-jpeg-dir=/usr/local/jpeg \
--with-png-dir \

--enable-mbstring=all \
	宽字节函数库对PHP的支持
	substr(字符串，开始位置，长度)--字符串截取，最小单位是字节
	mb_substr(字符串，开始位置，长度，字符集编码)--根据字符集编码会把每个实体内容看成一个单位，不会对汉字截取乱码
--enable-mbregex \
	正则表达式对PHP的支持
--enable-shared
