具体的奖金分配计划是这样的，孔蒂本赛季在联赛中共使用了23名球员，每个人的奖金平均在21.7万英镑左右，像卡希尔、阿斯皮利奎塔与阿扎尔几乎场场都首发的主力球员，奖金相应会得到更多一些。另外10%是留给替补球员以及没有机会出场的球员，包括特里、贝戈维奇、巴舒亚伊这样的球员，以保证给每个为切尔西做贡献的球员，他们也是团队中重要的一部分。此外，孔蒂和他的教练团队会得到一笔额外的奖金，并不包含在这500万英镑之内。具体的奖金分配计划是这样的，孔蒂本赛季在联赛中共使用了23名球员，每个人的奖金平均在21.7万英镑左右，像卡希尔、阿斯皮利奎塔与阿扎尔几乎场场都首发的主力球员，奖金相应会得到更多一些。另外10%是留给替补球员以及没有机会出场的球员，包括特里、贝戈维奇、巴舒亚伊这样的球员，以保证给每个为切尔西做贡献的球员，他们也是团队中重要的一部分。此外，孔蒂和他的教练团队会得到一笔额外的奖金，并不包含在这500万英镑之内。
