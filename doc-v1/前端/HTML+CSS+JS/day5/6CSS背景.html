虽然远隔万里

虽然远隔万里，但中国和斐济及其它南太平洋岛国有着悠久的民间交往历史，华人早在160多年前就已踏足斐济，目前该国80多万人口中华人人口已超过一万，在斐济社会中扮演着重要角色。中斐自1975年建交以来高层互访频繁，特别是习近平主席于2014年11月对斐济进行了国事访问，两国建立起了互相尊重、共同发展的战略性伙伴关系，对此，姆拜尼马拉马总理在接受采访时首先给予了高度评价。“自1975年两国建交以来斐中关系取得了强有力的发展，特别是习主席2014年来访，我本人也在2015年对中国进行了回访。可以说，双边关系的发展越来越强劲有力。”虽然远隔万里，但中国和斐济及其它南太平洋岛国有着悠久的民间交往历史，华人早在160多年前就已踏足斐济，目前该国80多万人口中华人人口已超过一万，在斐济社会中扮演着重要角色。中斐自1975年建交以来高层互访频繁，特别是习近平主席于2014年11月对斐济进行了国事访问，两国建立起了互相尊重、共同发展的战略性伙伴关系，对此，姆拜尼马拉马总理在接受采访时首先给予了高度评价。“自1975年两国建交以来斐中关系取得了强有力的发展，特别是习主席2014年来访，我本人也在2015年对中国进行了回访。可以说，双边关系的发展越来越强劲有力。”
