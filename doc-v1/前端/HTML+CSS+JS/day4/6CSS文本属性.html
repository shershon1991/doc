巴基斯坦总理为中方建设的高速公路项目点赞

国际在线报道（中国国际广播电台记者 王琦）：10日，巴基斯坦总理谢里夫在巴国家公路局主席塔拉尔等陪同下，视察了中国铁建旗下中铁二十局集团承建的巴基斯坦卡拉奇至拉合尔高速公路第三标段项目，对项目建设情况表示满意。这是自2016年8月以来，谢里夫总理第二次视察该项目。国际在线报道（中国国际广播电台记者 王琦）：10日，巴基斯坦总理谢里夫在巴国家公路局主席塔拉尔等陪同下，视察了中国铁建旗下中铁二十局集团承建的巴基斯坦卡拉奇至拉合尔高速公路第三标段项目，对项目建设情况表示满意。这是自2016年8月以来，谢里夫总理第二次视察该项目。国际在线报道（中国国际广播电台记者 王琦）：10日，巴基斯坦总理谢里夫在巴国家公路局主席塔拉尔等陪同下，视察了中国铁建旗下中铁二十局集团承建的巴基斯坦卡拉奇至拉合尔高速公路第三标段项目，对项目建设情况表示满意。这是自2016年8月以来，谢里夫总理第二次视察该项目。
